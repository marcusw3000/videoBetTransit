# Frontend Oficial

Esta pasta (`frontend/`) é a fonte principal da interface web do projeto.

## Desenvolvimento

- `npm run dev`: sobe o frontend em modo local
- `npm run build`: gera o bundle de produção
- `npm run lint`: valida a base com ESLint

## Observação

A pasta `traffic-counter-front/` foi mantida temporariamente como referência legada. Novas mudanças devem ser feitas em `frontend/`.
